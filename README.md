# .simossd-notes

> Short, focused notes while learning cybersecurity.

---

A minimal documentation space dedicated to capturing cybersecurity concepts, tools, and techniques in a clear and concise form.

This repository is used as a structured reference for key ideas encountered during learning, broken down into simple notes for easier understanding and revision.

The goal is not depth in one place, but clarity across many topics, keeping everything lightweight, readable, and easy to revisit over time.

---

### ⚠️ Important Note

This documentation was created using Obsidian.

For the best experience, it is highly recommended to view and use these notes within Obsidian.  
Some features such as internal linking, graph view, formatting, and structure may not display correctly in standard text editors or directly on GitHub.

Using Obsidian ensures the notes appear as intended and preserves the full workflow and navigation system.
